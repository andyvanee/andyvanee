Copy Path to Clipboard

Mar 13, 2012

Finder makes it a bit difficult to get the full path to a file; like when you’re pasting it into config files or Terminal.

So here’s a little script that makes it this easy: Browse in Finder, right click a file or folder and choose ‘Copy Path to Clipboard’.
